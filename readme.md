# wndt 核心功能
0.基于wnd frontend
1.前端用户中心提供：注册登录、充值、站内信
2.付费阅读，付费下载，用户投稿并分成
3.无限可能性的后续功能开发

wndt 本身并不复杂，事实上它本质是对wnd-frontend插件的一次最简单的基础开发，是用主题的形式对插件功能的一次直观表达。真正的核心仍然在于wnd frontend
基于wnd frontend 可以轻松开发：客户管理系统，报名系统，在线教育站点等等。你可以不信，但我们正在这样做。
